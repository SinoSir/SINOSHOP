## 3. 黑盒 API 服务定义

以下服务通过 API 网关对外暴露，**仅提供接口，不暴露实现**。

### 3.1 BDMP 阻尼计算服务
- **端点：** `POST /api/v1/bdmp/compute`
- **功能：** 接收波浪参数与构件状态，返回阻尼力向量与能量消耗预估。
- **输入格式：** 见 `specs/r16_definition.proto` 中定义的 `BDMPRequest`
- **输出格式：** `BDMPResponse`

### 3.2 虎符令牌核验服务
- **端点：** `POST /api/v1/tiger/verify`
- **功能：** 验证虎符令牌的有效性、剩余额度与地理围栏匹配。
- **输入格式：** `TigerVerifyRequest`
- **输出格式：** `TigerVerifyResponse`

### 3.3 RWA 资产确权服务
- **端点：** `POST /api/v1/rwa/confirm`
- **功能：** 接收构件状态数据，触发链上资产确权与价值更新。
- **输入格式：** `RWAConfirmRequest`
- **输出格式：** `RWAConfirmResponse`

### 3.4 动态权重查询服务
- **端点：** `GET /api/v1/weight/{zone_type}`
- **功能：** 返回指定 ZON 类型的当前动态权重系数。
- **输出格式：** `WeightResponse`
