# 虎符令牌接口契约 / TIGER TOKEN INTERFACE

> **状态：** 公开接口定义 | **版本：** V1.0  
> 本文件定义虎符令牌的公开交互接口。  
> 令牌的生成、核验、多签授权等内部实现由 **SINOSHOP-Core** 管理，不属于本文件范围。

---

## 1. 概述

虎符令牌是物理构件申领、激活、绑定 R16 编码的唯一合法授权凭证。  
每一枚令牌，是四项信息的刚性绑定：

1. **持有单位与责任主体身份**（Holder Identity）
2. **授权可生产/注册的构件类型范围**（Component Types）
3. **限定物理作业地理围栏与海域边界**（Geo-Fence）
4. **编码申领、激活、备案的最大使用次数**（Quota）

令牌一经激活，四项信息 **不可更改**。
